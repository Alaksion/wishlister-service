## Agent skills

### Issue tracker

Issues and PRDs live as GitHub issues in `Alaksion/wishlister-service`. External PRs are **not** treated as a triage surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Uses the default canonical label vocabulary: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context repo. Read `docs/domain-model/domain-model.md` for domain context and `docs/adr/` for architectural decisions. See `docs/agents/domain.md`.
